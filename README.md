# ADK-Tutorial

Contained in this folder are the different AI agents created in Google's ADK tutorial, available [here](https://adk.dev/get-started/python/)

This workspace includes four top-level agent folders:

- `A_getting_started`: A basic introduction to agent creation and tool integration.
- `B_multi_tool_agent`: Demonstrates an agent that can use multiple tools in a single interaction.
- `C_agent_team`: Shows agent orchestration with sub-agents, session state, and guardrails.
- `D_streaming_agent`: Contains a streaming agent example, including a Google search agent implementation.

Look through the folders here in case any refresh is needed for different simple ADK topics.
